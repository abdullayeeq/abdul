# Mohammed Abdul Layeeq — Portfolio Website

A modern, futuristic AI/ML-themed portfolio website built with pure HTML, CSS, and JavaScript.

## 📁 Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── css/
│   ├── style.css       # Main stylesheet
│   └── animations.css  # Scroll animations & transitions
├── js/
│   ├── neural.js       # Canvas neural network background
│   └── main.js         # Interactions, typewriter, counters
└── resume.pdf          # Place your resume PDF here
```

## 🚀 Features

- **Animated neural network** canvas background (interactive with mouse)
- **Custom cursor** with magnetic hover effects
- **Typewriter animation** in hero section
- **Scroll reveal** animations on all sections
- **Animated counter** stats in About section
- **Skill bar** animations triggered on scroll
- **Project cards** with glow effects matching each project's color
- **Responsive design** for desktop, tablet, and mobile
- **Dark futuristic theme** with cyan/purple/blue accents
- **Glassmorphism** contact form

## 🛠️ How to Use

1. Open `index.html` in your browser — no build tools needed!
2. To add your profile photo: replace the `.profile-placeholder` div in `index.html` with an `<img>` tag pointing to your photo.
3. Add your `resume.pdf` to the root folder.
4. Update GitHub/LinkedIn URLs if needed.

## 🎨 Customization

- Colors: Edit CSS variables at the top of `css/style.css` (`:root` block)
- Fonts: Currently using **Syne** (headings) + **DM Sans** (body)
- Projects: Edit the project cards in `index.html`
- Skills: Update skill bars with `data-width` attributes

## 📱 Browser Support

Works in all modern browsers: Chrome, Firefox, Safari, Edge.
